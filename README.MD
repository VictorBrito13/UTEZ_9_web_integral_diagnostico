# Crypto Assets API Client

This project is a simple API client for a Crypto Assets API. It is a simple API client that allows you to create, read, update and delete crypto assets.

## Run the project

```bash
docker compose up --build
```

## Technologies

### Backend

- [FastAPI](https://fastapi.tiangolo.com/)
- [SQLAlchemy](https://www.sqlalchemy.org/)
- [PyMySQL](https://pymysql.readthedocs.io/en/latest/)
- [Pydantic](https://docs.pydantic.dev/)

### Frontend

- [Vite](https://vitejs.dev/)
- [TypeScript](https://www.typescriptlang.org/)

### Orchestrator

- [Docker](https://www.docker.com/)
- [Docker Compose](https://docs.docker.com/compose/)

### Database
- [MySQL](https://hub.docker.com/_/mysql)

## Features

- Create a crypto asset
- Read a crypto asset
- Update a crypto asset
- Delete a crypto asset
